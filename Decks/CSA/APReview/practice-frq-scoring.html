Question 1

Methods and Control Structures

7 points
(a) findWinner — 4 pts (b) shortenMessage — 3 pts

Part (a): findWinner

Simulate a race between carOne and carTwo over numLaps laps by using getLapTime(lapNum) for each car.

Compare the total time for both cars and return exactly one outcome string: "Car 1 wins!", "Car 2 wins!", or "Tie!".

Part (b): shortenMessage

Return a shortened version of message with every substring equal to str removed.

If str never appears, return the original message unchanged. The prompt guarantees the occurrences of str do not overlap.

Q1a · Point 1 of 4
findWinner
public String findWinner(int numLaps)
{
    double car1Time = 0;
    double car2Time = 0;

    for(int j = 1; j <= numLaps; j++)
    {
        car1Time += carOne.getLapTime(j);
        car2Time += carTwo.getLapTime(j);
    }

    if (car1Time < car2Time)
    {
        return "Car 1 Wins!";
    }
    else if(car2Time < car1Time)
    {
        return "Car 2 Wins!";
    }
    else
    {
        return "Tie!";
    }
}

shortenMessage
public String shortenMessage(
        String message, String str)
{
    String result = message;
    while (result.indexOf(str) >= 0)
    {
        int idx = result.indexOf(str);
        result = result.substring(0, idx)
               + result.substring(
                   idx + str.length());
    }
    return result;
}

Class Design

7 points

Complete a CubePair class built from two Cube objects. A Cube can be a number, letter, or color cube, and calling roll() returns one face label as a string.

Write two constructors: one that creates two cubes of the same type, and one that creates two cubes from two possibly different types. Then write rollCubes() so it rolls both cubes and returns the two results concatenated into one string.

Q2 · Point 1 of 7
CubePair
public class CubePair
{
    private Cube cube1;
    private Cube cube2;

    public CubePair(String type)
    {
        cube1 = new Cube(type);
        cube2 = new Cube(type);
    }

    public CubePair(String type1, String type2)
    {
        cube1 = new Cube(type1);
        cube2 = new Cube(type2);
    }

    public String rollCubes()
    {
        String result = "";
        result += cube1.roll();
        result += cube2.roll();
        return result;
    }
}

Data Analysis with ArrayList

5 points

Write getHighRangeVehicles for an EVDataSet that stores electric vehicles in vehicleList.

The method should return an ArrayList<String> containing the model names of every vehicle whose range is greater than target, but without duplicate model names. The original vehicleList must remain unchanged.

Q3 · Point 1 of 5
getHighRangeVehicles
public ArrayList<String> getHighRangeVehicles(int target)
{
    ArrayList<String> highRange =
        new ArrayList<String>();

    for (ElectricVehicle ev : vehicleList)
    {
        if (ev.getRange() > target)
        {
            boolean found = false;
            for (String s : highRange)
            {
                if (s.equals(ev.getModelName()))
                    found = true;
            }
            if (!found)
                highRange.add(ev.getModelName());
        }
    }
    return highRange;
}

Question 4

2D Arrays

6 points

Write numberOfPies for an AppleCrate that stores apples in a 2D array. Each pie needs poundsPerPie pounds of usable apples.

An apple cannot be used if it is rotten, and it also cannot be used if it is directly adjacent to a rotten apple. Find the total usable apple weight, then return the number of whole pies that can be made.

Q4 · Point 1 of 6
numberOfPies
public int numberOfPies(double poundsPerPie)
{
    double weight = 0;

    for (int r = 0; r < apples.length; r++)
    {
        for (int c = 0; c < apples[0].length; c++)
        {
            boolean bad = apples[r][c].isRotten();
            if (r > 0 &&
                apples[r-1][c].isRotten())
                bad = true;
            if (r < apples.length-1 &&
                apples[r+1][c].isRotten())
                bad = true;
            if (c > 0 &&
                apples[r][c-1].isRotten())
                bad = true;
            if (c < apples[0].length-1 &&
                apples[r][c+1].isRotten())
                bad = true;
            if (!bad)
                weight += apples[r][c].getWeight();
        }
    }
    return (int)(weight / poundsPerPie);
}

Accesses all elements of apples (correct loop bounds across both dimensions).

Can still earn
  • Accesses elements out of bounds during adjacency checks, as long as the main loop bounds still traverse every element
Will NOT earn
  • Fails to access elements of apples correctly (e.g., wrong loop bounds that miss elements)
